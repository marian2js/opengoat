# @opengoat/ui

Optional UI extension for OpenGoat.

- React + Vite frontend
- Fastify server
- Single port for API + UI (default `19123`)

## Scripts

```bash
pnpm --filter @opengoat/ui dev
pnpm --filter @opengoat/ui build
pnpm --filter @opengoat/ui start
```

## Runtime

`OPENGOAT_UI_PORT` sets the server port (default: `19123`).
