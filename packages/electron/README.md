# @opengoat/desktop

Electron desktop app for OpenGoat.

## Stack

- Electron Forge + Vite
- React + Tailwind
- shadcn-style component primitives
- Zustand state management
- tRPC over IPC (`electron-trpc`)

## Run

From repo root:

```bash
pnpm install
pnpm desktop:dev
```

## Package

```bash
pnpm desktop:package
pnpm desktop:make
```
