## Global Flags

OpenGoat supports these process-level options before commands:

- `--version` / `-v`
- `--log-level <silent|error|warn|info|debug>`
- `--log-format <pretty|json>`

Example:

```bash
opengoat --log-level info --log-format pretty agent list
```
